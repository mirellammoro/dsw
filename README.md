# dsw
Dataset Showcase Workshop

Repositório de arquivos do SBBD DSW -- Dataset Showcase Workshop do Simpósio Brasileiro de Bancos de Dados.

Aqui você encontra o modelo para submissão de artigos e um exemplo de sua utilização:

* main.tex: exemplo de utilização do modelo, incluindo novas funções como ORCID
* sbc-template.bib: exemplos de referências bibliográficas, incluindo para repositório de dados no Zenodo
* sbc-template.sty: LaTeX definitions for SBC 2001 style (v2017)
* sbc.bst: modelo para entradas bibliográficas, incluindo DOI e respectivo link

Se quiser usar o Overleaf para visualizar os arquivos, clique [aqui](https://www.overleaf.com/read/kxtgxqckbpwm#c414e3).

Outras informações sobre o workshop estão disponíveis [aqui](https://sites.google.com/view/sbbd-dsw)
